# Multi-Doc-RAG
Multi-Doc-RAG is an advanced Retrieval-Augmented Generation (RAG) system designed to enable intelligent question answering across multiple documents. It combines semantic search with large language models (LLMs) to generate accurate, context-aware responses from heterogeneous data sources.

Unlike traditional QA systems, this project retrieves relevant chunks from multiple documents and augments the LLM’s response with grounded context, reducing hallucinations and improving factual accuracy
